🐍 Praca z plikami w Pythonie

Kompletny przewodnik po metodach odczytu i zapisu

📄 Metoda .read()

with open("demo.txt", "r") as plik:
    tresc = plik.read()
    print(tresc)

Co to robi?

  • Wczytuje **całą zawartość** jako jeden tekst.
  • Dobre dla małych plików.

📏 Metoda .readline()

with open("demo.txt", "r") as plik:
    linia1 = plik.readline()
    print(linia1)

Co to robi?

  • Czyta tylko **jedną, kolejną linię**.
  • Zwraca tekst wraz ze znakiem końca linii \n.

📚 Metoda .readlines()

with open("demo.txt", "r") as plik:
    lista_linii = plik.readlines()
    print(lista_linii[0])

Co to robi?

  • Wczytuje wszystko i robi z tego **listę** linii.
  • Możesz odwołać się do konkretnego numeru linii.

🧹 Metoda .strip()

with open("demo.txt", "r") as plik:
    for linia in plik:
        czysta_linia = linia.strip()
        print(czysta_linia)

Po co to robimy?

  • Pliki tekstowe na końcu każdej linii mają ukryty znak \n (enter).
  • strip() usuwa ten znak oraz zbędne spacje z początku i końca tekstu.
  • Dzięki temu unikasz podwójnych odstępów przy robieniu print().

✂️ Metoda .split()

dane = "Jan;Kowalski;25"
elementy = dane.split(";")

print(elementy) 
# Wynik: ['Jan', 'Kowalski', '25']

Jak to działa?

  • Tnie jeden długi tekst na mniejsze kawałki (listę) tam, gdzie znajdzie wybrany znak.
  • Jeśli nie podasz znaku w nawiasie, split() potnie tekst po spacjach.
  • Idealne do czytania tabelarycznych danych z plików TXT lub CSV.

📝 Tryb zapisu ("w")

with open("wynik.txt", "w") as plik:
    plik.write("To jest pierwsza linia\n")
    plik.write("To jest druga linia")

Tryb "w" (write):

  • Uwaga: Jeśli plik już istniał, ten tryb go całkowicie czyści i zapisuje od nowa!
  • Jeśli plik nie istnieje, Python go stworzy.
  • Pamiętaj o dodaniu \n na końcu tekstu, jeśli chcesz zacząć od nowej linii.

➕ Tryb dopisywania ("a")

with open("logi.txt", "a") as plik:
    plik.write("Dodaję kolejną wiadomość...\n")

Tryb "a" (append):

  • Nie kasuje starej zawartości – dodaje nowy tekst na samym końcu pliku.
  • Idealne do robienia list wyników lub logów.

📋 Zapisywanie całej listy

owoce = ["jabłko", "banan", "cytryna"]

with open("lista.txt", "w") as plik:
    for owoc in owoce:
        plik.write(owoc + "\n")

Szybki trik:

  • Funkcja write() przyjmuje tylko napisy (stringi).
  • Jeśli chcesz zapisać liczbę, musisz ją zamienić: plik.write(str(123)).

🔢 Macierze (Listy wielowymiarowe)

Jak przechowywać dane w formie tabeli

🏗️ Tworzenie macierzy

# Macierz 3x3 (lista list)
macierz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

Co to jest?

  • Macierz w Pythonie to po prostu **lista, która zawiera inne listy**.
  • Każda wewnętrzna lista to jeden **wiersz** macierzy.

🎯 Odwoływanie się (Indeksowanie)

# Wybranie liczby 6 (wiersz 1, kolumna 2)
element = macierz[1][2]
print(element) # Wynik: 6

Jak to czytać?

  • Pierwszy nawias [1] to numer **wiersza**.
  • Drugi nawias [2] to numer **kolumny**.
  • Liczymy zawsze od **0**!

🔄 Przechodzenie pętlą (for)

# Najprostszy sposób przejścia
for wiersz in macierz:
    for element in wiersz:
        print(element, end=" ")
    print()

Jak to działa?

  • Pierwszy for bierze cały wiersz (listę).
  • Drugi for wyciąga liczby z tego wiersza.
  • print() na końcu robi nową linię dla wyglądu.

🌍 Zmienne Globalne i Lokalne

Gdzie żyją Twoje dane i kto ma do nich dostęp

⚖️ Globalna vs Lokalna

x = "Globalna"

def moja_funkcja():
    y = "Lokalna"
    print(x) # Widzi globalną
    print(y) # Widzi lokalną

moja_funkcja()

Jaka jest różnica?

  • Globalna: Dostępna wszędzie.
  • Lokalna: Działa tylko w środku funkcji.

🔑 Słowo kluczowe 'global'

licznik = 0

def zwieksz():
    global licznik 
    licznik += 1

zwieksz()
print(licznik)

Po co to?

  • Użyj global, jeśli funkcja ma **zmienić** coś na zewnątrz.
  • Bez tego Python zrobi nową zmienną wewnątrz funkcji.

📋 Listy i ich funkcje

Podstawowe operacje na zbiorach danych

➕ Dodawanie i Usuwanie

owoce = ["jabłko", "banan"]

owoce.append("pomarańcza") # Dodaje na koniec
owoce.insert(1, "mango") # Wstawia na pozycję 1

owoce.remove("banan") # Usuwa konkretny element
ostatni = owoce.pop() # Usuwa i zwraca ostatni element

Najważniejsze funkcje:

  • append(): Najczęstszy sposób na powiększenie listy.
  • insert(indeks, wartosc): Pozwala wcisnąć coś w środek.
  • pop(): Przydatne, gdy chcesz coś zabrać z listy i od razu tego użyć.

🔍 Informacje i Sortowanie

liczby = [5, 2, 8, 1, 9]

dlugosc = len(liczby) # Ile jest elementów? (5)
liczby.sort() # Układa od najmniejszej
liczby.reverse() # Odwraca kolejność

if 5 in liczby:
    print("Jest piątka!")

Przydatne sztuczki:

  • len(): Podstawa przy pętlach for.
  • sort(): Zmienia listę na stałe (układa ją).
  • Słowo in: Błyskawicznie sprawdza, czy coś jest w środku.

📝 Funkcja .append()

lista_gosci = []

while len(lista_gosci) < 3:
    imie = input("Podaj imię: ")
    lista_gosci.append(imie)

print(lista_gosci)

Dlaczego append?

  • To podstawowa funkcja do budowania dynamicznych list.
  • Zawsze dodaje element na **szary koniec** listy.
  • W słownikach nie używamy append – tam po prostu przypisujemy nową wartość do klucza.

📖 Słowniki (Dictionaries)

Przechowywanie danych za pomocą kluczy

🔑 Podstawy Słownika

osoba = {
    "imie": "Jan",
    "wiek": 25,
    "miasto": "Warszawa"
}

print(osoba["imie"]) # Wynik: Jan
osoba["zawod"] = "Programista" # Dodawanie nowej pary

Jak to działa?

  • Słownik tworzymy za pomocą nawiasów klamrowych { }.
  • Zamiast numerów (indeksów), używamy własnych nazw (kluczy).
  • Każdy klucz musi być unikalny.

🔄 Pobieranie danych przez klucze

osoba = {"imie": "Jan", "wiek": 25}

# 1. Pobranie listy kluczy
klucze = osoba.keys()

# 2. Wykorzystanie kluczy do pobrania wartości
for k in klucze:
    wartosc = osoba[k]
    print("Klucz:", k, "| Wartość:", wartosc)

Jak to działa?

  • osoba.keys() daje Ci listę wszystkich "etykiet" (np. imię, wiek).
  • W pętli for zmienna k przyjmuje po kolei każdy klucz.
  • Używając osoba[k], pytasz słownik: "co masz schowane pod tym konkretnym kluczem?".

🔍 Sprawdzanie czy klucz istnieje

menu = {"pizza": 30, "burger": 25}

danie = "frytki"

if danie in menu:
    print("Cena wynosi:", menu[danie])
else:
    print("Brak dania w karcie!")

Jak to działa?

  • Instrukcja if klucz in slownik sprawdza tylko **klucze**, a nie wartości.
  • Jest to bardzo szybka operacja, niezależnie od wielkości słownika.
  • Dzięki temu unikasz błędu, gdybyś próbował odwołać się do czegoś, czego nie ma.

✂️ Slicing (Wycinanie list)

Magiczne sztuczki z nawiasami kwadratowymi

🔄 Odwracanie: [::-1]

tablica = [1, 2, 3, 4, 5]

odwrocona = tablica[::-1]
print(odwrocona) # [5, 4, 3, 2, 1]

napis = "Kajak"
print(napis[::-1]) # "kajaK"

Jak to działa?

  • Trzeci parametr to **krok**.
  • Wartość -1 mówi Pythonowi: "idź od końca do początku o jeden krok".
  • To najszybszy sposób na sprawdzenie palindromu (czy słowo czyta się tak samo wspak).

🔪 Wycinanie fragmentów

liczby = [0, 1, 2, 3, 4, 5, 6, 7]

print(liczby[1:4])  # [1, 2, 3] (od 1 do 3)
print(liczby[:3])   # [0, 1, 2] (od początku do 2)
print(liczby[5:])   # [5, 6, 7] (od 5 do końca)
print(liczby[::2])  # [0, 2, 4, 6] (co drugi element)

Zasada [start:stop:step]:

  • Start: od którego indeksu zaczynamy (włącznie).
  • Stop: na którym kończymy (**bez niego!**).
  • Step: co ile elementów przeskakujemy.

🔐 Kod ASCII i Szyfrowanie

Przekształcanie znaków na liczby i proste szyfry

🔢 Funkcje ord() i chr()

kod = ord("A")   # Wynik: 65
znak = chr(66)   # Wynik: "B"

# Przesunięcie literki o 1
nowa = chr(ord("a") + 1) # Wynik: "b"

Jak to działa?

  • ord() zamienia pojedynczą literę na jej numer w tabeli ASCII.
  • chr() robi odwrotnie – zamienia numer na znak.
  • Pamiętaj: Małe 'a' (97) i wielkie 'A' (65) mają inne numery!

🔄 Szyfr ROT13 (Analiza)

def rot13(slowo):
    zamiana = ""
    for i in range(len(slowo)):
        kod = ord(slowo[i]) + 13
        
        if kod > 122: # Wyjście poza 'z'
            kod -= 26
        zamiana += chr(kod)
    return zamiana

Co tu się dzieje?

  • Przesuwamy każdą literę o 13 miejsc w alfabecie.
  • if kod > 122: Jeśli przesunięcie wyjdzie poza 'z', odejmujemy 26 (liczbę liter w alfabecie), żeby wrócić na początek (do 'a').
  • To prosty sposób na ukrycie treści – ponowne użycie ROT13 na zaszyfrowanym tekście go odszyfruje!

⚙️ Konwersja Systemów Liczbowych

Zamiana napisów na liczby dziesiętne

🎯 Funkcja int() z podstawą

# Z binarnego (2) na dziesiętny
bin = int("1010", 2)  # Wynik: 10

# Z szesnastkowego (16) na dziesiętny
hex = int("1A", 16)    # Wynik: 26

# Z ósemkowego (8) na dziesiętny

Parametry funkcji int():

  • Pierwszy argument to **tekst** (liczba zapisana jako string).
  • Drugi argument to **podstawa systemu**, z którego tłumaczysz.
  • Działa to dla dowolnego systemu od 2 do 36!

💻 Kod

Pełny skrypt rozwiązujący problem obiegu pakietów

📄 Skrypt Python

# Wczytywanie danych z pliku
plik = open('/odbiorcy_przyklad.txt', 'r')
wiersze = plik.readlines()
odbiorcy = []

for wierz in wiersze:
    wierz = int(wierz.strip())
    odbiorcy.append(wierz)

# Przygotowanie listy pakietów
test = [4, 3, 5, 3, 1, 2]
pakiety = []
for i in range(len(test)):
    pakiety.append(i + 1)

# Definicja funkcji runda
def runda(lista):
    pakiet_kopia = lista.copy()
    for i in range(len(test)):
        lista[i] = pakiet_kopia[test[i] - 1]
    return lista

# Główna logika szukania rundy (Twoja pętla while)
r = 0
kiedy = True
while kiedy == True:
    pomoc = runda(pakiety)
    r += 1
    for i in range(len(pomoc)):
        if pomoc[i] == i + 1:
            print(r, i + 1)
            kiedy = False
            break

plik.close()

💡 Wskazówka: Ten kod symuluje przemieszczanie się wartości wewnątrz listy na podstawie indeksów zdefiniowanych w tabeli test. Pętla while szuka momentu, w którym wartość pod danym indeksem jest równa samemu indeksowi (powiększonemu o 1).

🚀 Analiza

Szczegółowe wyjaśnienie logiki przesyłania pakietów

💻 Kod Python

def runda(lista):
    pakiet_kopia = lista.copy()
    for i in range(len(test)):
        lista[i] = pakiet_kopia[test[i] - 1]
    return lista

r = 0
kiedy = True
while kiedy == True:
    pomoc = runda(pakiety)
    r += 1
    for i in range(len(pomoc)):
        if pomoc[i] == i + 1:
            print(r, i + 1)
            kiedy = False
            break

🔄 Jak działa funkcja runda?

To serce algorytmu, które odpowiada za jeden krok symulacji:

  • lista.copy(): Tworzy "zamrożony" stan pakietów. Dzięki temu wszystkie paczki są wysyłane jednocześnie, a nie jedna po drugiej.
  • test[i] - 1: Twoja "mapa" połączeń. Wskazuje, który komputer jest odbiorcą pakietu z obecnej pozycji.
  • Przesunięcie: Kod bierze pakiet z kopii i wkłada go do lista na miejsce docelowe.

🎯 Wykrywanie Powrotu

Jak program znajduje odpowiedź?

  • Licznik r: Zlicza każdą wykonaną pełną turę przesyłek.
  • Test Powrotu: if pomoc[i] == i + 1. Sprawdzasz, czy numer pakietu zgadza się z numerem komputera, w którym aktualnie leży.
  • Flaga kiedy: Użyta jako wyłącznik bezpieczeństwa. Gdy znajdziesz pierwszy taki przypadek, zmieniasz ją na False, co natychmiast przerywa pętlę while.

🚀 Maturalne Must-Have

Algorytmy, które oszczędzają czas na egzaminie

🔢 Systemy i bity

liczba = 100

# Ile jedynek w zapisie binarnym?
jedynek = bin(liczba).count("1")

# Zamiana z dowolnego systemu (np. 3)
dziesietna = int("210", 3)

# Szybkie potęgowanie modulo (częste!)
wynik = pow(a, b, m) # (a^b) % m

Dlaczego to ważne?

  • count() w połączeniu z bin() pozwala błyskawicznie rozwiązywać zadania o "liczbach binarnych".
  • pow(a, b, m) jest kluczowe, gdy liczby są tak duże, że kalkulator wysiada.

📊 Sortowanie złożone

# Sortowanie listy słowników (np. uczniów)
uczniowie = [
    {"imie": "Anna", "pkt": 90},
    {"imie": "Jan", "pkt": 95}
]

# Sortuj według punktów (malejąco)
uczniowie.sort(key=lambda x: x["pkt"], reverse=True)

Jak to działa?

  • key=lambda pozwala Pythonowi "zajrzeć" do środka słownika lub krotki podczas sortowania.
  • To najszybszy sposób na rankingi w zadaniach z plikami.

🧮 Matematyka i Typy

a = 17 // 3  # Dzielenie całkowite -> 5

Zastosowanie:

🔄 Pętle for i while

# Pętla for z continue
for i in range(5):
    if i == 2:
        continue # Przeskocz 2 i idź dalej
    print(i)

# Pętla while
licznik = 5
while licznik > 0:
    licznik -= 1

Różnica:

  • continue: Kończy bieżące powtórzenie i od razu zaczyna następne.
  • break: Całkowicie przerywa pętlę.
